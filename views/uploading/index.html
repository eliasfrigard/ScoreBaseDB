Instructions

1. Input should be in English (except title/location/region). Only English country names will be accepted.

1. Specify only one value in each field (except for Collections and Tags).

2. Uploaded files should be .mxl (compressed XML) format only.

3. Song Key should be "A, D, Cm, F#m" etc. If you want to specify e.g. "Lydian", do so in the Tags section.

4. Title, Composer, Song Type, Location, Region & Country can be max 50 characters long (including whitespace).

5. Collections speicify the origin of the tune, e.g. "Vanhoja Pelimannisävelmiä" or "Svenska Låtar".

6. Additional information should be placed in the Tags section.

7. Collections and Tags can take multiple values, which should be separated by commas.

8. If you make a mistake or want to change something, contact the admin.
